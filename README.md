# Installation
Having installed [rcm](https://github.com/thoughtbot/rcm), proceed as follows:

```bash
git clone https://github.com/paulo-e/dotfiles ~/.dotfiles
sh .dotfiles/update.sh
sh .dotfiles/done.sh
```
